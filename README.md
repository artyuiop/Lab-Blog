# Lab PHP Admin Blog

## รายละเอียดโปรเจกต์
นี่คือ **ระบบ Blog แบบง่าย** ที่พัฒนาด้วย PHP สำหรับส่งงานแลป/โปรเจกต์มหาวิทยาลัย

- เฉพาะ **ผู้ดูแลระบบ (Admin)** ที่สามารถเข้าสู่ระบบเพื่อสร้าง แก้ไข หรือลบบทความได้  
- **ผู้ใช้งานทั่วไป** สามารถดูบทความที่เผยแพร่แล้วเท่านั้น  
- พัฒนาด้วย **PHP**  

---

## ฟีเจอร์

### สำหรับ Admin
- Login / Logout
- Create Post (สร้างบทความ)
- Edit Post (แก้ไขบทความ)
- Delete Post (ลบบทความ)
- View All Posts

### สำหรับผู้ใช้งานทั่วไป
- ดูบทความทั้งหมดที่เผยแพร่
- อ่านรายละเอียดบทความแต่ละเรื่อง
